Labyrinth
=========

Main Menu
---------

P – Play 
S – Go to the scoreboard
Esc – Exit the game

Gameplay
--------


Main Movement
- Arrow Keys to move table

Camera Controls
- Pivot camera up : 8
- Pivot camera down : 2
- Pivot camera left : 4
- Pivot camera up : 6

- Zoom in : +
- Zoom out : -

Go to main menu : M
Reset Game: P
View Top Scores: B
