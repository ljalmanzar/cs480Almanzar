Labyrinth
=========

Main Movement
- Arrow Keys to move table

Lighting Controls
- For ambient lighting : A
- For spot light : S
- For distant light : D
- For point light : F

Camera Controls
- Pivot camera up : 8
- Pivot camera down : 2
- Pivot camera left : 4
- Pivot camera up : 6

- Zoom in : +
- Zoom out : -

Go to main menu : M
Reset Game: P
